## 🎯 핵심 기능

### 반응형 디자인
- 데스크톱: 최대 너비 800~1200px 제한
- 모바일: 세로 1열 레이아웃
- 터치 친화적 버튼 크기

### 사용자 경험
- 카드 기반 디자인 (그림자 + 둥근 모서리)
- 파란색 (#2196F3) 포인트 컬러
- 명확한 CTA 버튼
- 이메일 복사 기능 (SnackBar 피드백)

### 네비게이션
- go_router 사용한 SPA 라우팅
- 깔끔한 URL 구조
- 뒤로가기 버튼 지원

## 📦 사용된 패키지

- `go_router`: 라우팅
- `url_launcher`: 외부 링크 (앱스토어)
- `google_fonts`: Noto Sans KR 폰트
